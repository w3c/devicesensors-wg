The group has a publicly archived mailing list <public-device-apis@w3.org> that is mainly used for announcements. Technical discussions happen in the GitHub repositories that host the specifications. Please see the roadmap and follow the GitHub links.

Member-confidential messages and logistical discussions can be addressed to the member-only archived mailing list <member-device-apis@w3.org>

IRC channel #dap on irc.w3.org port 6665 (Web IRC client). This channel is mainly used during teleconferences and F2F meetings.
